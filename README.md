# SmartCardio2 - Next.js App

### Установка, сборка и запуск

`npm install`

`npm run build`

`npm start`


### Интеграция со СДЭК

- Все API-запросы к проксируются через `CDEK_BASE_URL`.
- При старте сервера `instrumentation.ts` запускается `startTokenRefreshLoop()`.
- Токен получается через `/v2/oauth/token` и сохраняется в файле `token.json`.
- Токен автоматически обновляется каждые 30 минут.
- Клиент не видит токен, т.к. используются серверные API вида (`/api/cdek/*`).

| Вызываемые API:                | Описание                  |
|--------------------------------|---------------------------|
| `GET /api/cdek/cities?q=`      | Автодополнение городов    |
| `GET /api/cdek/pvz?city_code=` | Список ПВЗ в городе       |
| `POST /api/cdek/calc`          | Расчёт стоимости доставки |
| `POST /api/cdek/order`         | Создание заказа           |

### Промокоды

- Пользователь вводит текст в поле «Промокод» на шаге подтверждения (русский язык поддерживается).
- Через 600 мс после последнего ввода фронтенд делает запрос на **серверный** роут `GET /api/promocodes/check?code=CODE`.
- Сервер проксирует запрос к `PARTNER_API/api/promocodes/findByCode?code=CODE` с заголовком `X-API-KEY`.
- Если ответ содержит `id` - промокод валиден. Поле окрашивается зелёным, цена пересчитывается с учётом `discountPercent`.
- **Важно:** скидка применяется только к отображаемой покупателю цене. В заказ СДЭК передаётся полная стоимость товара, чтобы комиссия СДЭК считалась от неё корректно.

| Вызываемые API:                                    | Описание                                             |
|----------------------------------------------------|------------------------------------------------------|
| `GET /api/promocodes/check?code=`                  | Проверка промокода                                   |
| `POST /api/sales/create?promocodeId={id}`          | Регистрация продажи по промокоду (после заказа СДЭК) |

После успешного создания заказа СДЭК, если был введён валидный промокод, сервер автоматически вызывает `PARTNER_API/api/sales/create?promocodeId={id}` с заголовком `X-API-KEY`. Ошибка этого вызова не блокирует оформление заказа.
